# expo-api

Application which lets user to connect to
 - Youtube
 - Twitch
 - Medium
 - Instagram
 - Pinterest
 - Spotify

## Run the project

```
    git clone https://github.com/maxgfr/expo-api.git
    cd expo-api
    mv .env.example .env  # set your own credentials
    npm install
    expo start
```
